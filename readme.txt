This is a simple plugin to show the related posts based on category and / or tags.

== Description ==

This Plugin allows you to show the related post as per your choice. You have the flexibility to select posts based on the same category or tag or both.
Maxim number of posts and heading are customizable.  

= Main Features =

* Simple and user friendly interface.
* No programming knowledge required to implement.
* No header or function files modification required.
* Fully customizable.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/related-posts-by-aarvis` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the "Related Posts - Aarvis" link on the admin screen to configure the plugin.
